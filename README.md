# JobNova Landing

A modern job search landing page built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**.

## Features

- **AI-Powered Job Details Panel** – Intelligent job insights and recommendations
- **Responsive Layout** – Mobile-first design with sidebar navigation and drawer support
- **Modern UI Components** – Custom icons, like button interactions, and app shell architecture

## Tech Stack

| Category | Technology |
|----------|------------|
| Framework | Next.js 16 |
| UI Library | React 19 |
| Styling | Tailwind CSS 4 |
| Language | TypeScript |
| Testing | Vitest + React Testing Library |

## Getting Started

```bash
# Navigate to the project
cd jobnova-landing

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |
| `npm run test` | Run tests |
| `npm run test:watch` | Run tests in watch mode |
